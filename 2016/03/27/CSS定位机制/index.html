CSS定位机制

  • css

display:block特征

  • 默认宽度为父元素宽度
  • 可设置宽高
  • 换行显示
    常见元素有：div,p,h1-h6,ul,form

display:inline特征

  • 默认宽度为内容宽度
  • 不可设置宽高
  • 同行显示
    常见元素有：span,em,a,label,cite

display:inline-block特征

  • 默认宽度为内容宽度
  • 可设置宽高
  • 同行显示
    常见元素有：input,button,textarea,select

position:relative特征

  • 默认宽度为父元素宽度
  • 仍然在文本流中
  • 参照物为元素本身
    经常用于绝对定位元素的参照物，因为其本身设置relative不会对布局有任何影响。

position:absolute特征

  • 默认宽度为内容宽度
  • 脱离文本流
  • 参照物为第一个定位祖先元素或根元素

position:fixed特征

  • 默认宽度为内容宽度
  • 脱离文本流
  • 参照物为视窗

float:left/right特征

  • 默认宽度为内容宽度
  • 半脱离文本流
  • 向指定方向一致移动（知道不能移动为止）

box-sizing: content-box(默认) | border-box(IE)
